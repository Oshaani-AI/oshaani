About Public Share URLs

What is a public share URL?

  • A shareable link that anyone can use to access your agent
  • No email required - just share the URL
  • Perfect for sharing on social media, websites, or with large groups
  • Users will need to log in (or create an account) to use the agent

Security:

  • You can deactivate the URL at any time
  • Set an expiration date for automatic deactivation
  • Track how many times the URL has been accessed
  • Users can only use the agent, not modify it
