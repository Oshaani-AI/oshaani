Usage Instructions
Agent Chat (Requires Agent API Key)

Use the agent's API key for chat operations with that specific agent.

curl -X POST http://localhost:8000/api/agents/7/chat/ \
  -H "Content-Type: application/json" \
  -H "Authorization: ApiKey AGENT_API_KEY" \
  -d '{"message": "Hello"}'
MCP Server (Requires User API Key)

Use your user API key for MCP operations and general API access.

curl -X POST http://localhost:8080/mcp \
  -H "Content-Type: application/json" \
  -H "Authorization: ApiKey YOUR_USER_API_KEY" \
  -d '{"jsonrpc":"2.0","id":1,"method":"tools/list","params":{}}'
Python Example - Agent Chat
import requests

# Use AGENT API KEY for agent chat
headers = {
    "Authorization": "ApiKey AGENT_API_KEY",
    "Content-Type": "application/json"
}

response = requests.post(
    "http://localhost:8000/api/agents/7/chat/",
    headers=headers,
    json={"message": "Hello"}
)
